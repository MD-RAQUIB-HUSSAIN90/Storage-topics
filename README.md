# JavaScript Storage APIs Practice

This repository contains my hands-on practice and learning exercises for JavaScript Storage APIs.

## Topics Covered

### Local Storage

* setItem()
* getItem()
* removeItem()
* clear()
* Storing Strings
* Storing Objects using JSON.stringify()
* Retrieving Objects using JSON.parse()

### Session Storage

* setItem()
* getItem()
* removeItem()
* clear()
* Session-based data storage

### Cookies

* Creating Cookies
* Reading Cookies
* Updating Cookies
* Deleting Cookies
* Cookie Expiration
* Cookie Attributes

## Learning Goals

* Understand client-side data storage.
* Learn the differences between Local Storage, Session Storage, and Cookies.
* Practice storing, retrieving, updating, and deleting data.
* Build a strong foundation for frontend development and real-world web applications.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Dom manipulation
